This website allows you to upload an image and generate a cross-stitch pattern based on the image's colors and size. You can adjust the size of the pattern and limit the number of color groups used.
Once the pattern and key are generated, you will be able to download them. They are deleted from the server after 24 hours, so I recommend saving them.
